Net2Source - Charlotte, NC
Net2Source, Inc. is one of the fastest growing IT Consulting company across USA. N2S is headquartered at NJ, USA with its branch offices in Asia Pacific Region. N2S offers a wide gamut of consulting solutions customized to client needs including staffing, training and technology.

Title: Oracle DBA
Location: Charlotte, NC
Duration: 6+ months

Job Description: -
  • 24 * 7 Supports.
  • Experience resolving Oracle Network I/O such as SQL Net, ODBC,LDAP,JNDI and MPL
  • Strong knowledge in Oracle agent, GI ,Golden Gate Replication
  • Strong knowledge in Exadata or Exadata knowledge is desired
  • Closely work with Vendors, Stay Up-to date with Product Features
  • Expertise in Application Partitioning, Information Remnance,Denial of Service Protection and Boundary Protection mechanisms Administers and ensure proper statistics all database objects
  • Troubleshoot RMAN Performance, OS Huge Pages ,OS Kernel Settings, Identify and Resolve Disk I/O Hotspots and fragmentation
  • Troubleshoots SQL Query Performance Issues, problem scenarios, on Oracle Identity Manager, OID and Database Vault
  • Perform Database Upgrade to new Oracle version, quarterly PSU upgrades.
  • Effective interpersonal skills, enhancing Partners to use newer trends in technology and SQL Best Practices.
  • Knowledge of Logical and Physical Database Design, assisting Teams to adhere Governance during Implementation.
  • Strong skills in Oracle Database Performance using AWR,ADDM,ASH,SQLT, Oracle Data UNLOADER
  • Demonstrate excellent knowledge and practical skills in Oracle 12c AWR Warehouse and ASH Analytics
  • Strong Understanding of Oracle Optimizer, optimizer parameters, parsing mechanism and how optimizer influences in controlling, stabilizing execution plans
  • Accomplished in handling Performance Benchmarking and configuration, of in-memory database tuning and other middleware.
  • Strong Knowledge in Predictive Index Reorganization utilizing Index Advisory and Reducing LIOs
  • Thorough knowledge of Performance Reporting using OEM Grid 12c and BI Publisher dashboards
  • Thorough knowledge of Oracle Code Action and benchmarking in Massively Parallel Systems
  • Application Code Reviews in PL/SQL, Performance Testing using Oracle Database Replay and using SQL Performance Analyzer
  • Ability to detect and resolve system waits involving cache buffer chains, detecting benefits from super-sizing of PGA
  • Thorough knowledge of Dimensional Data Modeling and traditional ER modeling and leveraging Performance Impacts on DWH systems
  • Strong Performance Tuning Skills in Distributed Global Data warehouse Architecture, Data Marts and Operational Data Stores.
